Detailed Description

Base class of the temporary expression.

TempExprs are pass specific expression that can be useful to define intermediate result in the rewriting pass such as layout or type transformation.

Subclass TempExprNode allows us to pattern match on specific kind of TempExpr and use them for expression rewriting.

TempExpr should only be used within a pass,
