\subsection{Detailed Description}
A variable node in the IR. 

A variable is uniquely identified by its address.

Each variable is only binded once in the following nodes\+:
\begin{DoxyItemize}
\item Allocate
\item For
\item Let
\item Let\+Stmt 
\end{DoxyItemize}
